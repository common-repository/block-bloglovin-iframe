Stop your blog from showing inside of the Bloglovin iframe. Instead, visitors will be redirected to your actual blog post.

== Description ==

By default, [Bloglovin](https://bloglovin.com) will display your blog posts within their iframe. This plugin will block this from happening by automatically redirecting the iframe to the original link on your blog.

== Frequently Asked Questions ==

= How do I install this plugin? =

1. Go to 'Plugins > Add New' in your WordPress dashboard and search for "bloglovin iframe".
2. Install and activate the plugin. that's it!

If you wish to stop using the plugin you can deactivate it at any time.

= How does it work? =

This plugin will add some code to your site which automatically redirects any iframes to the original source.

= Will my blog still be on Bloglovin? =

Yes your blog will continue to work on Bloglovin. This plugin will only remove the Bloglovin iframe from showing at the top of your posts and content.
